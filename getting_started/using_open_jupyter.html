Jupyter Open¶

We will be using UBC’s version of JupyterHub. This runs in your web browser and supports the latest versions of Chrome, Edge, Firefox, or Safari.

In a web browser, go to: https://open.jupyter.ubc.ca/ and log in with your UBC CWL.

Select Python Notebook, and click ‘start’

You will download labs and worksheets from canvas, and upload them to your space on Jupyter Open - see Week 1 instructions on canvas for links and more details.
